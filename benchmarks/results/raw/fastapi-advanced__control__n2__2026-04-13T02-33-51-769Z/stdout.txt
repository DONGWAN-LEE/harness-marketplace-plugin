All 3 tests pass. Here's a summary of what was implemented:

**`app/auth.py`**
- `_b64url_encode` / `_b64url_decode` — base64url helpers using stdlib `base64`
- `_sign` — HMAC-SHA256 via stdlib `hmac` + `hashlib`, keyed on `settings.jwt_secret`
- `create_access_token(user_id)` — builds `header.payload.signature` JWT with `exp = now + access_token_expire_minutes`
- `verify_token(token)` — splits token, constant-time signature check via `hmac.compare_digest`, expiry check, returns `int(sub)`
- `get_current_user_id` — FastAPI dependency with optional `Authorization` header; raises `HTTPException(401)` if missing, malformed, or invalid

**`app/routes/me.py`**
- `GET /me` — uses `Depends(get_current_user_id)`, returns `{"user_id": <id>}`

**`app/main.py`**
- Added `app.include_router(me.router)`

**`tests/test_auth.py`**
- 401 on missing header (was returning 422 before fixing dependency to `Header(default=None)`)
