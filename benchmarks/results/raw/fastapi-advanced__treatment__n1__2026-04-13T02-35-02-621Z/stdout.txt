All checks pass. Here's a summary of what was implemented:

**`app/auth.py`**
- `_b64url_encode` / `_b64url_decode` — stdlib base64url helpers
- `_sign(header_b64, payload_b64)` — computes `HMAC-SHA256` over `header.payload` using `settings.jwt_secret`
- `create_access_token(user_id)` — builds a signed HS256 JWT with `sub`/`exp` claims
- `verify_token(raw)` — splits, checks signature with `hmac.compare_digest`, checks expiry, returns `user_id`; raises `HTTPException(401)` on any failure
- `get_current_user_id(authorization)` — FastAPI dependency; accepts `Authorization: Bearer <jwt>`, returns the decoded `user_id`

**`app/routes/me.py`** — `GET /me` protected by `Depends(get_current_user_id)`, returns `{"user_id": <id>}`

**`app/main.py`** — `me.router` registered

**`tests/test_auth.py`** — three tests covering missing header (401), valid bearer (200 + correct user_id), and an expired JWT (401)
