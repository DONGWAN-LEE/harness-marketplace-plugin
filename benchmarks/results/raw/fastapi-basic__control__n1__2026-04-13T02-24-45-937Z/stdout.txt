All acceptance criteria met. Here's a summary of what was created:

**`app/schemas/user.py`** — Pydantic v2 `UserResponse` with `from_attributes = True` for ORM compatibility.

**`app/schemas/__init__.py`** — Empty init to make it a package.

**`app/routes/users.py`** — `GET /users/{user_id}` using `Depends(get_db)`, returns 200 with `UserResponse` or 404 with `{"detail": "User not found"}`.

**`app/main.py`** — Router registered with `app.include_router(users.router)`.

**`tests/conftest.py`** — Sets `DATABASE_URL` to `sqlite+pysqlite:///:memory:` (Python's built-in driver) before any app module is imported, so the engine instantiation in `database.py` doesn't require `psycopg`.

**`tests/test_users.py`** — Uses `app.dependency_overrides[get_db]` to inject a mock session that returns `None`, verifying the 404 path without a real database.
