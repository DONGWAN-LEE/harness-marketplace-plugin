The file `app/profile/edit/page.tsx` has been created. Here's what it implements:

**Structure:**
- `'use client'` directive — required since the component uses `useState` and event handlers
- `FormValues` / `FormErrors` interfaces for typed form state

**Validation:**
- `validateEmail` — regex check (`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`)
- `validate` — checks name (min 2 chars) and email format; returns an errors object
- Errors are set on submit; inline error messages render below each field via conditional `<p role="alert">` elements

**Behavior:**
- On submit with invalid data: inline errors shown, no success message
- On submit with valid data: `success` state set, "Profile updated" renders below the form
- Editing a field clears the success banner

**TypeScript:** All event handlers are typed (`React.ChangeEvent<HTMLInputElement>`, `React.FormEvent<HTMLFormElement>`), state is generically typed — no `any` usage, compatible with strict mode.
